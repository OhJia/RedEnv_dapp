What is a red envelope?

It's an East Asian tradition to gift envelopes containing money to family and friends for celebrations. The color red symbolizes good luck, so sending money wrapped in red is a way to bestow good fortune to the receiver.


How does Lucky Envelope work?

Create and send an envelope: Go to Create and set the amount of Ether and set a passcode. Then send the URL and passcode of this envelope to everyone you want to gift.


Claim an envelope: Go to the URL you received and enter the passcode. Tap "claim" to receive a portion of the original deposit.
